# CanDogmatic
Som principalment un lloc web basat en notícies, opinions i anàlisis que se centra en els musulmans d'arreu del món i, en particular, en els països de parla espanyola, catalana i europea, però que també al món islàmic en general. Ens mantindrem independents i professionals, centrats en l'actualitat, el diàleg interreligiós i els dubtes moderns.
